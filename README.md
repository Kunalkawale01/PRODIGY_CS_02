# PRODIGY_CS_02

Develop a simple image encryption tool using pixel manipulation. You can perform operations like swapping pixel values or applying a basic mathematical operation to each pixel. Allow users to encrypt and decrypt images.

ImagePixelEncryption/
├── main.py
├── crypto.py
├── README.md
└── requirements.txt

## How the Project Works (Simple)

Uses pixel manipulation (XOR operation) on RGB values

Same numeric key is used for encryption & decryption

Supports PNG / JPG images

Encryption is reversible and fast

## How to Run

1. Extract the ZIP

2. Open terminal in the folder

3. Install dependency:

       pip install pillow

4. Run:

       python main.py
